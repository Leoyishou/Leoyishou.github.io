Java 8 introduced the Stream API, which provides a set of features for performing operations on sequences of elements, such as data manipulation, data extraction, transformation, filtering, sorting, and others.

Here are some examples to illustrate how to use Java 8 streams:

1. Creating a Stream:

To use streams, first, you need a source to create a Stream. Here is an example of creating a Stream from a Collection (List):

2. Filtering:

This is an example of using a stream to filter elements. The filter operation takes a Predicate to test each input element.

3. Mapping:

The map operation applies a given function to each element.

4. Collecting Results:

You can use the collect method to store the elements back in a list or another data structure.

5. Reduction:

The reduce operation performs a reduction on the elements of the stream with the given function.

6. Parallel streams:

In Java 8, the concept of parallel streams was introduced, which utilizes multiple cores of your processor to execute the operations in parallel to improve performance.

These are just some of the ways you can use Java 8’s Stream API. The API provides a range of other operations, such as flatMap, distinct, limit, peek, and so on.
