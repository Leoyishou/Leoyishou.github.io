Obsidian Custom Sort

Created2023-06-18|Updated2023-06-18|ToolsObsidian
|Post View:
how can I sort files in obsidian as the order I want?
Obsidian, a powerful knowledge base on top of a local folder of plain text Markdown files, offers various ways to organize and sort your notes, but it doesn't natively support custom ordering of files. Here are a few workarounds you could consider:

  1. File Naming: One way to sort files in a particular order is to prepend the filenames with a numbering or dating scheme, for instance: 001-Note, 002-Note, etc., or 2021-06-18-Note, 2021-06-19-Note, etc. Files in Obsidian will then be sorted alphabetically, and thus in the order you want.

  2. Folders: You can create folders to categorize your notes and keep them organized. This won’t allow for a custom order within the folder, but it will help keep related notes together.

  3. Plugins: Obsidian has a strong community of developers who create plugins to extend the functionality of the tool. There may be a plugin that provides more advanced sorting or ordering options. You can explore these in the Community Plugins section of Obsidian.

  4. Embedding in a note: You can create a main note where you link to all other notes in the order you want. The advantage of this method is that you can also add context or a summary to each link, creating a kind of table of contents or dashboard for your notes.

Remember to always back up your notes regularly, as Obsidian works with your local files and changes you make in Obsidian will affect the files on your computer.

search related plugins for me
Based on my search, there are two plugins that can help you sort files in Obsidian in the order you prefer:

  1. Obsidian Custom Sort by SebastianMC: This plugin allows you to take full control over the order and sorting of folders and notes in Obsidian’s File Explorer. You can freely arrange notes and folders, and it supports both automatic and manual ordering. You can set the custom sorting order for a specific folder by creating a new note named sortspec in the folder and putting the sorting specification in the YAML front matter of the note. The plugin also supports inheritance of order specifications with flexible exclusion and overriding logic​1​.

  2. Obsidian File Order: This plugin allows you to reorder files with drag-and-drop by customizing a number-prefix in the filenames. After installation, you can right-click on any folder or item and click on “Reorder items” to open the reorder dialog. In the reorder dialog, you can drag and drop items to customize their order. You can also clear a custom ordering you made in the past, which will remove the prefixes from the file names and restore the default ordering. The plugin maintains no settings itself, every ordering information used is stored in the file names themselves. In the plugin settings, you can also define default values for prefix length, delimiter and starting index​2​.

Both of these plugins can be found and installed from the official Obsidian Community Plugins page.
